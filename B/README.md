# B: BASIC

The B-drive contains BASIC, in this case:

* MBASIC.COM
  * Microsoft BASIC v5.29 [28-Jul-85]
  * Enter "SYSTEM" to exit.
* OBASIC.COM
  * An older Microsoft BASIC, v5.21 [28-Jul-81]
  * Enter "SYSTEM" to exit.
* TBASIC.COM
  * [Tasty BASIC](https://github.com/dimitrit/tastybasic)
  * Enter "BYE" to exit.
